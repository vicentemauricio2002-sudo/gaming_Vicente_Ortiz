# Página Gaming

## Descripción
Esta página web nos adentra a la industria de los videojuegos, tanto Nintendo como Rockstar Games han marcado generaciones y en esta página exploraremos dos de sus títulos más famosos.
Esta es una versión con acciones correctivas y evolutivas de la página web Gaming anterior.

## Mejoras realizadas
+ Eliminación de estilos en línea: Todo cambio de estilo del HTML es definido en un CSS externo.
+ Mejora de tipografía y estructura: Uso del estilo en CSS para una mejor organización de la información en la página, tanto en la fuente como en la separación de secciones.
+ Ajuste de imágenes: Usando CSS se cambió el tamaño de las imagenes para tener más congruencia y un aspecto más ordenado.
+ Más interactividad usando hover: Con CSS se le otorgó más interactividad y dinámica a la navegación de la página con una versión mucho más visualmente atractiva.
